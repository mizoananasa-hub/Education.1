🎒 NEW STUDENT REGISTRATION FLOW
================================

Students should NO LONGER instantly create accounts.

Instead:

==================================================
📝 STEP 1 — STUDENT REQUEST FORM
================================

Student submits REQUEST form:

Fields:

* Full Name
* Grade
* Religion
* Optional parent phone/email

When submitted:
➡️ request goes to admin dashboard

Student DOES NOT get account immediately.

==================================================
👑 STEP 2 — ADMIN APPROVES STUDENT
==================================

Admin sees:
"Pending Student Requests"

Admin can:

* Approve
* Reject

==================================================
✅ WHEN APPROVED
===============

System automatically generates:

* Student Code
* Temporary Password

Example:
STU-1042

Password:
Learnova2026

Store in database.

==================================================
📨 STEP 3 — STUDENT RECEIVES ACCESS
===================================

After approval:
Student can now login using:

* student code
* password

OPTIONAL:
Show approval screen/message:
"Your request has been approved."
