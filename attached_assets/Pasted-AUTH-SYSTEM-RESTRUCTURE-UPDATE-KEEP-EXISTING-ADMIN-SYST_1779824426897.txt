⚠️ IMPORTANT:

* DO NOT rebuild the admin dashboard

==================================================
🔥 NEW AUTHENTICATION STRUCTURE
===============================

Students and teachers should now sign in using NORMAL account credentials.

REMOVE:

* subject password system
* hardcoded subject passwords
* subject password DB validation

Teachers should NO LONGER log in using:

* subject + subject password

==================================================
✅ NEW TEACHER LOGIN
===================

Teacher login fields:

* Email OR Username
* Password

Use:

* bcrypt hashed passwords
* JWT auth

==================================================
✅ NEW STUDENT LOGIN
===================

Student login fields:

* Student Code OR Email
* Password

Use:

* bcrypt hashed passwords
* JWT auth

==================================================
📝 SIGNUP SYSTEM WITH APPROVALS
===============================

Users can SIGN UP normally.

BUT:
accounts are NOT immediately active.

==================================================
🎒 STUDENT SIGNUP FLOW
======================

Student submits signup form:

Fields:

* Full Name
* Email (optional)
* Grade
* Religion
* Password
* Confirm Password

After signup:
➡️ account status = pending

Student CANNOT access dashboard until approved by admin.

==================================================

=========================

* Full Name
* Email

* Confirm Password
